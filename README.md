# stegg.dev Starter Template

> Part of the **Build Your Free Website in 3 Hours** course at [stegg.dev](https://stegg.dev)

## The only file you need to edit

```
src/data/content.js
```

Open that file, fill in your business details, save — your entire website updates automatically.

## Project structure

```
my-website/
├── index.html                  ← Entry point (don't touch)
├── style.css                   ← Global styles (don't touch)
├── src/
│   ├── app.js                  ← Wires everything together (don't touch)
│   ├── data/
│   │   └── content.js          ← ✏️ EDIT THIS FILE ONLY
│   └── components/
│       ├── Nav.js              ← Navigation bar
│       ├── Hero.js             ← Hero section
│       ├── About.js            ← About section
│       ├── Services.js         ← Services section
│       ├── Portfolio.js        ← Portfolio / gallery
│       ├── Testimonials.js     ← Customer reviews
│       ├── Contact.js          ← Contact + WhatsApp
│       └── Footer.js           ← Footer
```

## How to use in StackBlitz

1. Click the StackBlitz link provided in the course
2. Open `src/data/content.js`
3. Fill in your business name, tagline, services, WhatsApp number
4. Save (Ctrl+S) — preview updates live on the right
5. Push to GitHub → deploy on Cloudflare Pages

## Deploying to Cloudflare Pages

1. Push your project to GitHub from StackBlitz
2. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
3. Click **Create a project** → Connect to Git
4. Select your repo → click **Save and Deploy**
5. Your site is live at `yourname.pages.dev` 🎉
